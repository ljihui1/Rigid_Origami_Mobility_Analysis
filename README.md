# Rigid_Origami_Mobility_Analysis
MATLAB codes for mobility analysis of rigid origami

Reference:
Jihui Li, Jinye Zhu, Shaoxing Qu,
"A general second-order mobility analysis framework to reveal non-intuitive kinematic characteristics of rigid origami".
in submiting.
Copyright (c) 2026, Jihui Li, Jinye Zhu, Shaoxing Qu.

=============================================================================
S1 Miura_singleVertex: Run "main_singleVertexMiura_mobilityAnalysis.m" to obtain the second-order compatibility equations of single-vertex Miura-ori; run "main_singleVertexMiura_motionSimulation.m" to trace motion branches of single-vertex Miura-ori at the flat deployed state.

S2 Miura_twoVertex_row: Run "main_twoVertexMiura_mobilityAnalysis.m" to obtain the second-order compatibility equations of the Miura-ori with two vertices arranged in a row; run "main_twoVertexMiura_motionSimulation.m" to trace its different motion paths at the flat deployed state.

S3 Miura_threeVertex_column: Run "main_threeVertexMiura_mobilityAnalysis.m" to obtain the second-order compatibility equations of the Miura-ori with three vertices arranged in a column.

S4 FoldableOctahedron: Run "main_octahedron_mobilityAnalysis_oneDOF.m" to obtain the second-order compatibility equations of the foldable octahedron; run "main_octahedron_motionSimulation_oneDOF.m" to trace its motion path.

S5 Dodehedron-derived 8R origami_2DOFs: Run "main_DD8R_mobilityAnalysis.m" to obtain the second-order compatibility equations of the Dodehedron-derived 8R origami; run "main_DD8R_motionSimulation.m" to trace its motion paths.

S6 Dodehedron-derived 7R origami_bifurcation: Run "main_DD7R_mobilityAnalysis.m" to obtain the second-order compatibility equations of the Dodehedron-derived 7R origami; run "main_DD7R_motionSimulation.m" to trace its different motion paths at the bifurcation point.

S7 Dodehedron-derived shaky origami: Run "main_dodecahedron_mobilityAnalysis_rigid.m" to obtain the second-order compatibility equations of the Dodehedron-derived shaky origami.

S8 Diamond_singleVertex: Run "main_diamond_mobilityAnalysis.m" to obtain the second-order compatibility equations of the single-vertex diamond origami; run "main_diamond_motionSimulation.m" to trace its different motion paths at the bifurcation point.
